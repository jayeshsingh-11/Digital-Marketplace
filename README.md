This is a Digital Marketplace built with Next.js 14, Payload CMS, tRPC, and Tailwind CSS.

## Features
- Custom Categories
- Seller Dashboard
- Cart & Checkout
- Admin Dashboard
- Authentication (Email + Password)

## Getting Started

1. Install dependencies:
   ```bash
   yarn install
   ```

2. Run the development server:
   ```bash
   yarn dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
